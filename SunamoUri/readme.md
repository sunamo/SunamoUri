### SunamoUri

A .NET library for working with URLs (Uniform Resource Locators / Identifiers). Provides comprehensive URI manipulation, encoding/decoding, query string parsing, and URL sanitization utilities.

Part of PlatformIndependentNuGetPackages:

- [nuget.org](https://www.nuget.org/profiles/sunamo)
- [github.org](https://github.com/sunamo/PlatformIndependentNuGetPackages)

Another links:

- [Developer site](https://sunamo.cz)

Request for new features / bug report / etc: [Mail](mailto:radek.jancik@sunamo.cz) or on GitHub

## Main Components

- **QSHelper** - Query string parsing, building, and manipulation
- **UH** - URI manipulation, encoding/decoding, path extraction, URL sanitization

## Target Frameworks

**TargetFrameworks:** `net10.0;net9.0;net8.0`
